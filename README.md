## 简介

Wlftool是一个小的Java工具类库，让Java更方便使用。

-------------------------------------------------------------------------------

## 包含组件

一个Java基础工具类，对文件等JDK方法进行封装，组成各种Util工具类，同时提供以下组件：

| 模块            | 介绍          |
|---------------|-------------|
| wlftool-excel | 提供excel读写方法 |

可以根据需求对每个模块单独引入，也可以通过引入`wlftool-all`方式引入所有模块。
